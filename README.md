# @roucoule/send

> **Under construction** — this package is in early development.

`@roucoule/send` is an open-source CLI and library that lets you send
[Roucoule](https://github.com/thomascarvalho/roucoule) newsletters directly
through your own SMTP server — no third-party delivery service required.

## Features (planned)

- Send a Roucoule newsletter issue via any SMTP server
- Works as a standalone CLI (`roucoule-send`) or as a TypeScript/Node library
- Publishable to both JSR (`@roucoule/send`) and npm (`@roucoule/send`)
- Zero runtime dependencies beyond Deno std and nodemailer (Node build)

## Status

This package is **under active construction**. The API and CLI interface are not
yet stable. Follow progress at
[github.com/thomascarvalho/roucoule](https://github.com/thomascarvalho/roucoule).
